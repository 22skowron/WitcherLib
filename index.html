WitcherLib: Documentation¶

Release 0.0.1 (Installation).

WitcherLib is a fictional Python library inspired by The Witcher universe. It allows you to:

  • Summon Witchers

  • Brew potions

  • Hunt monsters

  • Interact with a simple Witcher world simulation

This documentation is intended to guide developers and fans of the series in using the library for fun projects or learning purposes.
